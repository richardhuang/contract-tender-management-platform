# 第三方合同和招标管理平台 - 系统文档

## 1. 项目概述

第三方合同和招标管理平台是一个全面的企业级解决方案，专门用于管理合同、招标和供应商关系。该平台提供了一套完整的工具来简化企业的采购流程，提高效率和透明度。

## 2. 核心功能

### 2.1 合同管理
- **合同创建**: 支持在线创建和编辑合同内容
- **合同审批**: 多级审批流程，支持会签和或签
- **合同归档**: 自动归档已完成的合同
- **合同跟踪**: 监控合同执行情况，设置关键节点提醒
- **合同变更**: 记录合同修订历史
- **合同到期提醒**: 提前通知即将到期的合同

### 2.2 招标管理
- **招标发布**: 创建招标项目，发布招标公告
- **投标管理**: 接收和管理投标文件
- **评标系统**: 结构化评标，支持多维度打分
- **开标管理**: 在线开标功能
- **中标通知**: 自动发送中标/未中标通知

### 2.3 供应商管理
- **供应商注册**: 供应商在线申请注册
- **资质审核**: 验证供应商资质信息
- **绩效评估**: 根据历史合作情况评估供应商表现
- **供应商分类**: 按行业、规模等对供应商分类
- **黑名单管理**: 对违规供应商进行标记

## 3. 系统架构

### 3.1 技术栈
- **前端**: React.js + Ant Design + React Router
- **后端**: Node.js + Express.js
- **数据库**: PostgreSQL + Sequelize ORM
- **认证**: JWT + bcrypt
- **文件存储**: AWS S3/MinIO

### 3.2 微服务架构
1. 用户服务 - 用户管理、认证和授权
2. 合同服务 - 合同生命周期管理
3. 招标服务 - 招标流程管理
4. 供应商服务 - 供应商信息管理
5. 通知服务 - 邮件、短信等通知
6. 存储服务 - 文档上传和存储
7. 报告服务 - 生成统计和分析报告

## 4. 数据库设计

### 4.1 核心实体
- **用户表 (Users)**: 系统用户信息
- **合同表 (Contracts)**: 合同基本信息
- **招标表 (Tenders)**: 招标项目信息
- **供应商表 (Vendors)**: 供应商信息
- **投标表 (Bids)**: 投标信息
- **审批流程表 (ApprovalWorkflows)**: 多级审批流程
- **审批阶段表 (ApprovalStages)**: 审批具体环节

### 4.2 关系设计
- 用户 - 合同: 一对多
- 用户 - 招标: 一对多
- 招标 - 投标: 一对多
- 供应商 - 投标: 一对多

## 5. 安全性

### 5.1 数据安全
- 敏感数据加密存储
- HTTPS/TLS传输层安全
- 文件上传验证
- SQL注入防护

### 5.2 访问控制
- 基于角色的访问控制(RBAC)
- 详细的审计日志
- 会话管理和超时控制

1. 配置环境变量
2. 初始化数据库
3. 运行数据迁移
4. 启动服务

使用提供的docker-compose文件进行容器化部署。

## 7. API接口

### 7.1 认证接口
- POST /api/auth/register - 用户注册
- POST /api/auth/login - 用户登录
- GET /api/auth/profile - 获取用户信息

### 7.2 合同接口
- GET /api/contracts - 获取合同列表
- POST /api/contracts - 创建合同
- PUT /api/contracts/:id - 更新合同
- DELETE /api/contracts/:id - 删除合同

### 7.3 招标接口
- GET /api/tenders - 获取招标列表
- POST /api/tenders - 创建招标
- PUT /api/tenders/:id - 更新招标
- POST /api/tenders/:id/publish - 发布招标

### 7.4 供应商接口
- GET /api/vendors - 获取供应商列表
- POST /api/vendors - 创建供应商
- PUT /api/vendors/:id - 更新供应商
